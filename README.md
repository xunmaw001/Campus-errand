# Campus-errand
这款校园跑腿微信小程序，专为在校师生打造。打开微信，无需下载额外应用，就能发布各类跑腿需求。代取快递、食堂带饭、帮忙打印资料等事项一键下单，操作简单便捷。小程序智能匹配附近空闲的跑腿同学，快速响应任务。下单后可实时追踪进度，完成服务后还能评价反馈。既能让同学们解决生活琐事，节省时间精力，又为有空闲的同学提供勤工俭学机会，让校园生活更高效、更贴心 。
